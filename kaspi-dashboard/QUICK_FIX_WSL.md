# ⚡ Быстрое исправление WSL (30 секунд)

## Способ 1: Через PowerShell (самый быстрый)

1. Нажмите `Win + X`
2. Выберите **"Терминал (Администратор)"** или **"Windows PowerShell (Администратор)"**
3. Скопируйте и вставьте эту команду:
   ```powershell
   wsl --update
   ```
4. Подождите 1-2 минуты
5. Выполните:
   ```powershell
   wsl --shutdown
   ```
6. Перезапустите Docker Desktop

## Способ 2: Через командную строку

1. Нажмите `Win + R`
2. Введите: `cmd`
3. Нажмите `Ctrl + Shift + Enter` (запуск от имени администратора)
4. Выполните:
   ```cmd
   wsl --update
   ```
5. Подождите завершения
6. Перезапустите Docker Desktop

## После обновления

Проверьте версию:
```powershell
wsl --version
```

Затем запустите Docker и Supabase:
```bash
npm run docker:up
```
