# Устранение неполадок

## Если дашборд не работает или не отображается:

### 1. Перезапустите сервер разработки
```bash
# Остановите сервер (Ctrl+C)
# Затем запустите снова:
npm run dev
```

### 2. Очистите кэш Next.js
```bash
# Удалите папку .next
rm -rf .next
# Или на Windows:
rmdir /s .next

# Затем запустите снова:
npm run dev
```

### 3. Проверьте консоль браузера
- Откройте DevTools (F12)
- Перейдите на вкладку Console
- Проверьте наличие ошибок

### 4. Проверьте консоль сервера
- Посмотрите на вывод в терминале, где запущен `npm run dev`
- Проверьте наличие ошибок компиляции

### 5. Проверьте подключение к Supabase
- Убедитесь, что переменные окружения в `.env.local` настроены правильно
- Проверьте подключение через страницу "Проверка БД"

### 6. Проверьте подключение к Мой склад
- Убедитесь, что токен в `.env.local` настроен
- Проверьте подключение через страницу "Проверка Мой склад"

## Частые проблемы:

### Ошибка: "Module not found"
- Выполните: `npm install`

### Ошибка: "Cannot find module"
- Удалите `node_modules` и `.next`
- Выполните: `npm install`
- Запустите: `npm run dev`

### Страница не обновляется
- Очистите кэш браузера (Ctrl+Shift+R)
- Перезапустите сервер

### Стили не применяются
- Убедитесь, что Tailwind CSS правильно настроен
- Проверьте файл `tailwind.config.js`
- Перезапустите сервер
